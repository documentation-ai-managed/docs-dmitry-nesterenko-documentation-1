## Payload Structure

Understand the structure of webhook payloads.

```json
{
  "event": "document.updated",
  "timestamp": "2024-01-15T10:30:00Z",
  "data": {
    "documentId": "doc_123",
    "userId": "user_456",
    "changes": ["title", "content"]
  }
}
```

## Troubleshooting

Common webhook issues and how to resolve them.

<ExpandableGroup>
  <Expandable title="Delivery Failures" default-open="false">
    Check endpoint availability and response times.
  </Expandable>
  <Expandable title="Signature Verification" default-open="false">
    Ensure proper signature validation.
  </Expandable>
</ExpandableGroup>

| Issue | Symptom | Solution |
|-------|---------|----------|
| No deliveries | Webhook not triggering | Check event subscriptions |
| Invalid signature | Security error | Verify signing key |
| Timeout | Request fails | Increase timeout limit |
