## Basic Setup

Getting started with configuration is straightforward. Follow these steps to set up your documentation environment.

<Steps>
  <Step title="Initialize" icon="play">
    Create a new documentation project.
    <CodeGroup tabs="CLI,Web">
      ````bash
      npx dndoc init my-project
      ````
      ````javascript
      // Web initialization
      const project = new DNDoc.Project({
        name: 'my-project',
        theme: 'default'
      });
      ````
    </CodeGroup>
  </Step>
  <Step title="Configure" icon="settings">
    Set up your project settings.
  </Step>
  <Step title="Deploy" icon="rocket">
    Publish your documentation.
  </Step>
</Steps>

## Configuration File

The main configuration file controls all aspects of your documentation.

```json
{
  "name": "My Project Docs",
  "theme": "dark",
  "plugins": ["search", "analytics"],
  "api": {
    "baseUrl": "https://api.example.com",
    "timeout": 5000
  }
}
```
